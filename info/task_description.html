Palindrome is a number, word or text that reads the same in both directions. Your task is to write a function that checks the number, word or text if it is a palindrome. If the number, word or text palindrome return "true" otherwise "false". Learn more about palindromes you can in the article on Wikipedia. You should also take into account the presence of punctuation, spaces and case sensitive.

 

Input: Number, word or text.

Output: True or False
